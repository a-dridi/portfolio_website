Quick Customer Management

Java 13 JavaFX SQLite

A desktop application for the management of customers and invoices. Invoice PDFs can be created and sent to the customer with only one click.

Do Tasks Reach Goals

Java Android

An app that allows you to create tasks and task categories. You can track your tasks and you get notice if you have tasks that are overdue.

Administration software "Haushaltsverwaltung"

Java JSF Hibernate Primefaces HTML CSS

A Java web application that allows you to manage your company or personal data such as finances, books, videos and more.

The application can be selfhosted on your Tomcat server and used on your computer or smartphone. The application needs a Postgresql database and can be connect to your Nextcloud instance or any Webdav application.
