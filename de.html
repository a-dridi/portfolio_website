Responsiver Web-Shop

Ein leicht konfigurierbarer Webshop, der auf allen Geräten (Smartphone, Tablet und Desktop-PC) genutzt werden kann. Kunden können direkt mit Ihrem PayPal-Konto bestellen und bezahlen. Verwaltungssoftware "Haushaltsverwaltung"

Java JSF Hibernate Primefaces HTML CSS

Eine Java-Webanwendung, mit der Sie Ihre geschäftlichen oder persönlichen Daten wie z.B. Finanzen, Bücher, Videos und mehr verwalten können. Die Anwendung kann auf Ihrem Tomcat-Server selbst gehostet und kann überall übers Internet auf ihrem Computer oder Smartphone verwendet werden.

Die Anwendung benötigt eine Postgresql-Datenbank und kann eine Verbindung zu Ihrer Nextcloud-Instanz oder einer beliebigen Webdav-Anwendung herstellen.

Webshop für Reiseagenturen "Reisebuero"

Java JSF Hibernate HTML CSS

Diese Webanwendung verfügt über eine Registrierungs- und Benutzerverwaltungsfunktion, mit der Sie einen Überblick über Ihre potenziellen Kunden haben.

Über mich


Ich komme aus Österreich und habe eine IT-Schule absolviert. Ich hatte seit meiner Kindheit Interesse an IT und begann mit dem Programmieren und Erstellen mehrerer Webseiten. Ich habe Webseiten mit HTML und JavaScript erstellt, aber auch das CMS Joomla verwendet. Während meiner Ausbildung an der IT-Schule begann ich auch Java zu programmieren und lernte mehr über Informatik.

Neben der Erstellung von Webseiten und Applikationen, richte ich auch Webanwendungen / -dienste ein und verwalte Linux-Server in meiner Freizeit. Dazu gehören beispielsweise Postfix-Mail-Server, Web-Mail-Server, Monitoring-Server, OpenVPN-Server und vieles mehr. Einige meiner Projekte sind auf meiner Website verfügbar, auf der ich auch Tutorials und Informationen zu Technologien veröffentliche.

Eine meiner Lieblingsprogrammiersprachen ist Java, für die ich eine Zertifizierungsprüfung abgelegt habe. Ich habe die Java SE 8 Programmer Certification-Prüfung abgelegt und mit einem sehr positiven Ergebnis bestanden.
