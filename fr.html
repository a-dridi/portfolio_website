Mes Projets

Ceci est un aperçu de certaines de mes applications de bureau, smartphone et Web créées. Certaines de mes applications sont également disponibles sur Github.

Quick Customer Management

Java 13 JavaFX SQLite

Une application de bureau pour la gestion des clients et des factures. Les PDF de factures peuvent être créé et envoyé au client en un seul clic. Vous pouvez également créer et les ajouter à la facture. Cette application est disponible dans plusieurs langues.

Do Tasks Reach Goals

Java Android

Une application qui vous permet de créer des tâches et des catégories de tâches. Vous pouvez suivre vos tâches et vous serez averti si vous avez des tâches en retard. Toutes les tâches et les tâches terminées sont affichées dans d'autres points de menu. Atteignez vos objectifs qui peuvent être définis dans les paramètres de l'application.

Logiciel d'administration "Haushaltsverwaltung"

Java JSF Hibernate Primefaces HTML CSS

Une application web Java qui vous permet de gérer les données de votre entreprise ou vos données personnelles telles que les finances, les livres, vidéos et plus. L'application peut être auto-hébergée sur votre serveur Tomcat et utilisée sur votre ordinateur ou votre smartphone. Utilise JSF, JQuery et Primefaces. L'application nécessite une base de données Postgresql et peut être connectée à votre instance Nextcloud ou à toute application Webdav.

Number Trainer

Java Android

Entraînez votre mémoire avec les chiffres et essayez de vous souvenir du plus grand nombre possible. Le système de niveau de ce jeu vous permet d’obtenir un sens unique de la réussite à chaque fois que vous avez réussi en entrant les bons numéros. Votre statut de niveau sera automatiquement enregistré et la difficulté augmentera par rapport à votre niveau. Disponible sur le Google Play Store.

BMI Tracker

Python PyQt5

Cette application vous permet de suivre votre IMC et votre poids. Vous pouvez calculer votre IMC et vérifier si vous avez un IMC en bonne santé. Tous vos calculs peuvent être exportés vers un fichier PDF ou CSV.

Boutique en ligne pour les agences de voyage "Reisebuero"

Java JSF Hibernate HTML CSS

Une application Web JSF pour les agences de voyages. Les utilisateurs peuvent utiliser cette application Web pour vérifier les trajets proposés par votre entreprise et ils peuvent s'abonner au type de trajets qu'ils souhaitent pour obtenir des offres adaptées. Cette application Web dispose d’une fonction d’enregistrement et de gestion des utilisateurs qui vous permet de suivre vos clients pontential.

À propos de moi


Je viens d'Autriche et je suis diplômé d'une école d'informatique. Je m'intéressais à l'informatique depuis mon enfance et j'ai commencé par programmer et créer plusieurs sites Web. J'ai créé des sites Web avec HTML et JS, mais j'ai aussi utilisé le CMS Joomla. Pendant mes études à l’informatique, j’ai aussi commencé à coder en arrière-plan et j’en ai appris davantage sur l’informatique.

En plus de créer des sites Web et des applications, je configure également des applications / services Web et administre des serveurs Linux pendant mon temps libre. Ceux-ci incluent, par exemple, le serveur de messagerie Postfix, le serveur de messagerie Web, le serveur de surveillance, le serveur OpenVPN, etc. Certains de mes projets sont disponibles sur mon site Web, où je publie également des tutoriels et des informations relatives à la technologie.

L'un de mes langages de programmation préférés est Java, pour lequel j'ai passé un examen de certification. J'ai passé l'examen de certification de programmeur Java SE 8 et j'ai obtenu un résultat très positif.
Si vous voulez en savoir plus sur moi ou sur ce que je fais, contactez-moi ou visitez ma page Github.
